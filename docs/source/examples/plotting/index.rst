``matplotlib`` stuff I wish I knew sooner
=========================================

:meth:`matplotlib.axes.Axes.set_box_aspect`
    Set aspect ratio of an axes.
    As opposed to :meth:`matplotlib.axes.Axes.set_aspect`, this sets the
    physical aspect ratio within the figure, not the aspect ratio of the 
    data within the axes.
